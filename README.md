# QuarkusCepService

Repositório de prática de Quarkus. Consiste em um consumo de API através do quarkus, principios SOLID e injeção de dependência.
Exercício do [Prof. Manoel Campos](https://github.com/manoelcampos)

## Tecnologias
- Quarkus (Framework)
- Java (Linguagem de Programação)
- Lombok (Extensão)
